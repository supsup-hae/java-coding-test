# 소수 찾기

## 📌 문제 정보
- **문제**: [소수 찾기](https://www.acmicpc.net/problem/1978)
- **난이도**: Bronze 2
- **풀이 날짜**: 2024-01-14

## 👓 문제 설명
주어진 N개의 수 중에서 소수가 몇 개인지 찾아서 출력하는 프로그램을 작성해야 합니다. 소수는 1보다 큰 자연수 중 1과 자기 자신만을 약수로 가지는 수입니다[4].

## 🚫 제한사항
- N은 100이하의 자연수
- 주어지는 수는 1,000 이하의 자연수[1]

## 🖥️ 입출력 예
| 입력 | 출력 |
|------|------|
| 4<br>1 3 5 7 | 3 |

## 📝 입출력 예 설명
- 1은 소수가 아님
- 3, 5, 7은 소수
- 따라서 소수의 개수는 3개

```java
import java.util.Scanner;

public class Main {
	public static void main(String[] args) {
		Scanner scanner = new Scanner(System.in);
		int n = scanner.nextInt();
		int answer = 0;
		for (int i = 0; i < n; i++) {
			int count = 0;
			int value = scanner.nextInt();
			for (int j = 1; j <= value; j++) {
				if (value % j == 0) {
					count++;
				}
			}
			if (count == 2) {
				answer++;
			}
		}
		System.out.println(answer);
	}
}
```
